# 简体中文(基础文案)

[h1]IT'S THRILLER TIME!![/h1]

[b]不够刺激？还是想要更刺激？[/b]

一个有点特别的小型尸潮事件 Mod 来了哇~桀桀桀！

[img]https://i.imgur.com/JrKQCpX.gif[/img]

当 [b]Thriller 事件[/b]触发时，Michael Jackson 的经典歌曲《Thriller》会开奏，然后一只 cosplay 成 MJ、身穿经典红色服装的 DJ 丧尸会闪亮登场，并带领一小队丧尸向你靠近哦！

嘿！你以为这个DJ只是来跳舞的吗？错啦！它还有普通丧尸 [b]10 倍的生命值[/b]，并在登场后的 [b]15 秒内锁血自动回满状态[/b]哦！ 你要么尽快搞定它，要么把它留下来一起来Thriller Time！

[h2]主要特色[/h2]

[list]
[*]随机触发的 Thriller 小型尸潮事件，并播放特殊音乐
[*]身穿红色服装、拥有强化生命值和登场保护的 特殊丧尸；
[*]事件会吸引附近丧尸，并从外围追加丧尸群（可设置），提高尸群密度；
[*]事件影响范围内的丧尸会变成快步蹒跚者，并有可调概率变成奔跑者；
[*]夜晚是 Thriller Time：事件触发概率翻倍；
[/list]

[h2]True Actions Dancing 联动[/h2]

如果同时安装并启用 [b]True Actions Dancing[/b] 和本项目附带的联动模组，演出还会获得额外效果：

[list]
[*]DJ 靠近玩家时会跳舞；
[*]当你在尸群中激情战斗时，交战圈外围的丧尸可能加入演出，与 DJ一起表演；
[*]伴舞丧尸受攻击时会恢复战斗状态，立刻给你来个爱的拥抱❥(^_-)
[*]攻击 DJ 会让整个舞团暂时解散；
[*]当然，你也可以清理掉大多数丧尸，只留下 DJ，与它来一场面对面的亲密斗舞！
[/list]

[img]https://i.imgur.com/nV3NbEI.gif[/img]

[h2]沙盒设置[/h2]

[list]
[*][b]触发半径[/b]：3–60 格，默认 15 格。附近有丧尸进入这个范围时，Thriller 事件才有机会触发；
[*][b]变为奔跑者的概率[/b]：0–100%，默认 10%。事件范围内未变成奔跑者的丧尸会成为快步蹒跚者；
[*][b]伴舞丧尸数量上限[/b]：2–30，默认 6。演出预热阶段，每游戏分钟会从聚怪范围外围追加 2 只至该上限数量的丧尸，以增加尸群和潜在伴舞者的密度；
[*][b]事件每分钟触发概率[/b]：0–100%，默认 15%。附近存在丧尸时，每游戏分钟判定一次；夜晚概率翻倍。
[*][b]事件冷却[/b]：1–12 游戏小时，默认 2。一场演出顺利落幕后，需度过冷却期才可能迎来下一场。
[/list]

[h2]安装与兼容性[/h2]

[list]
[*]适用于 [b]Project Zomboid Build 42[/b]；
[*]基础模组可独立使用，不强制依赖其他 Mod；
[*]舞蹈联动需要同时启用 [b]True Actions Dancing[/b] 与 [b]Is Time To Thriller? - True Actions Dancing Addon[/b]；
[*]当前版本以单人游戏为目标，暂不建议用于多人服务器；
[*]与其他修改丧尸速度、生成机制或动画状态的 Mod 同时使用时，可能产生覆盖或冲突。
[/list]

[h2]玩法提示[/h2]

想看节目，可以把奔跑者概率设为 0；想体验真正的惊悚之夜，那就把它调高。

音乐响起时可别只顾着欣赏——DJ 带来的观众，可能比你预想的更多。
